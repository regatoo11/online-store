# معمارية المشروع | Project Architecture

## المبادئ الأساسية

- **Laravel 12** مع الالتزام بمعايير الإطار الرسمية.
- **كود نظيف**: Controllers خفيفة، منطق الأعمال في **Services**، والوصول للبيانات عبر **Repositories** عند الحاجة الفعلية فقط.
- **بدون Packages غير ضرورية**: أي Package خارجي يجب أن يكون له سبب واضح وموثّق هنا.
- **Migrations + Seeders + Factories** لكل Model.
- **Tests** للأجزاء الحرجة: المصادقة، السلة، الطلبات، الدفع، صلاحيات الأدمن.

## هيكل المجلدات المخطط

```
app/
├── Http/
│   ├── Controllers/        # Controllers خفيفة (Store + Admin)
│   ├── Requests/           # Form Requests للتحقق من المدخلات
│   └── Middleware/         # مثل: التحقق من صلاحية الأدمن، اللغة
├── Models/                 # Eloquent Models
├── Services/               # منطق الأعمال (CartService, OrderService, PaymentService...)
├── Repositories/           # طبقة الوصول للبيانات عند الحاجة
└── Providers/
```

## اللغات (i18n)

- العربية هي اللغة **الافتراضية** مع دعم RTL كامل، والإنجليزية لغة ثانوية.
- كل النصوص عبر ملفات `lang/ar` و `lang/en` - **ممنوع النصوص الثابتة (hardcoded)**.
- مبدّل لغة سيضاف في المرحلة 3 مع الواجهة.

## قرارات تقنية

| القرار | السبب |
|--------|-------|
| MySQL | مدعومة في استضافة Hostinger المشتركة |
| Session/Cache/Queue على قاعدة البيانات | لا يتوفر Redis في الاستضافة المشتركة |
| SQLite in-memory للاختبارات | سرعة وعزل كامل |
| بدون config منشورة | Laravel 12 يعتمد الافتراضيات + `.env` لتقليل الملفات |

## المراحل

1. ✅ **هيكل المشروع وقاعدة البيانات** - الأساس الحالي.
2. **المستخدمون والصلاحيات** - أدوار (عميل/أدمن) عبر عمود role + Middleware.
3. **المنتجات والتصنيفات** - تصنيفات هرمية، صور، مخزون، بحث، slugs.
4. **السلة** - سلة بقاعدة البيانات مع دعم الزوار (session).
5. **الطلبات** - Checkout وحالات الطلب ودورة حياته.
6. **الدفع** - COD + محافظ إلكترونية + انستا باي مع رفع سكرين شوت الإيصال وتحقق الأدمن. طبقة دفع قابلة للتوسع.
7. **لوحة الإدارة** - إدارة المنتجات والتصنيفات والطلبات والعملاء وإعدادات الدفع.
8. **SEO** - Meta tags, Open Graph, sitemap.xml, slugs نظيفة.
9. **الأداء** - Eager loading, Indexes, Cache للتصنيفات والإعدادات.
10. **الاختبارات والنشر** - استكمال الاختبارات + CI + دليل النشر على Hostinger.
